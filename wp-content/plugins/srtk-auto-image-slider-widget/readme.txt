A simple widget plugin to display images as slider which are hyperlinked.

== Description ==
A simple widget plugin based on JS to display images as slider which can be hyperlinked to URLs. The number of images in the slider can be configured based on the count provided and will automatically slide with a time delay of 5 seconds. These images can be hyperlinked individually during setting up the widget.



== Installation ==
This section describes how to install the plugin and get it working.

1. Simply Upload/Install the widget plugin and activate
2. Go to Appearance > Widget and find this widget
3. Drag the widget to your preferred location sidebar
4. Add the count for number of image in slider and hit save
5. Add the URLs for image and hyperlink as per the count and save again
6. Enjoy your widget

Note: Set the count to 0 and hit save, if you don\'t want any slider images.

== Frequently Asked Questions ==
= To change the time delay between images? =

By default the time interval is set to 5 seconds, which can be configured through the Javascript (.js) file within the plugin. In the next version, it\'ll be hopefully configurable.
